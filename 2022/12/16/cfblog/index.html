使用 CFBlog 免费搭建博客

发表于2022-12-16|更新于2024-02-14|教程
|浏览量:

介绍

这是一个运行在cloudflare workers 上的博客程序,使用 cloudflare KV作为数据库,无其他依赖. 兼容静态博客的速度,以及动态博客的灵活性,方便搭建不折腾.

搭建

你需要准备两样东西：Cloudflare 账号和一个域名（必须绑定到 Cloudflare ）

首先，注册一个 Cloudflare 账号（已经有账号的可以跳过），之后点击这里进入控制台，点击左侧的Workers - KV，再点击创建命名空间，名称可以随便填，之后点击添加。

然后点击左侧的Workers，点击创建服务，服务名称可以随便改，其他保持不动，点击设置 - 变量，在下面的 KV 命名空间绑定里，点击添加绑定，变量名称填CFBLOG，KV 命名空间选择刚才创建的那个，然后点击保存并部署。

点击上面的快速编辑，将框里的代码全部删除，将这里的代码粘贴进去，按照代码里的注释按需求改动。

*区域 ID 在你要绑定的域名 - 概述的最下面。点击这里进入 API 令牌 创建页面，点击创建令牌，在创建自定义令牌一栏点击开始使用，如下图配置。

配置完后一路继续，最后得到 API 令牌。

全部改完后，点击保存并部署，回到主页，点击你要绑定的域名，点击DNS，添加一条解析记录，如下图添加。

填好后，点击保存，然后回到 Workers ，点击你的项目，点击触发器 - 添加路由，路由填写你 配置的域名/*，区域选择你绑定的域名，此时你就能使用自己的域名访问了。
