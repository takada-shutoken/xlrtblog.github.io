Hexo 静态博客搭建教程 - 配置 Waline 评论系统

发表于2022-12-04|更新于2024-02-14|教程
|浏览量:

前言

由于 Hexo 不自带评论系统，所以我们要自己配置评论系统。这里使用 Waline 评论系统进行配置。

Waline 简介

一款简洁、安全的评论系统。

搭建

LeanCloud 配置

点击这里注册 LeanCloud 国际版。

注册并且验证邮箱后，点击“创建应用”，应用名称随便填，应用的计价方案选择“开发版”，完成后点击“创建”。

进入你创建的应用，点击左侧菜单的“设置”，然后点击“应用凭证”，你会看到 AppID、AppKey 和 MasterKey，把他们记录下来，一会儿要用。

Vercel 配置

点击这里进入 Vercel 部署界面（没有帐户请先选择 Continue with GitHub ），Repository Name 随便填，然后点击 “Create”，等待部署完成。

部署完成后，点击“Continue to Dashboard”，之后点击上面的“Setting”，然后点击“Environment Variables”，连续点击两下“Add another”，接下来按下图配置。

配置完后，点击“Save”保存，然后点击上面的“Deployments”，之后点击右侧的三个点，点击“Redeploy”，然后再次点击“Redeploy”来重新推送。

注：以下操作可能需要挂梯子

等待STATUS变成Ready后，点击右侧“Visit”，之后在链接后加上 /ui/register 来注册管理员帐户。至此，Waline 评论系统搭建完成。

引入 Hexo

这里以 Fiuld 主题为例，其他主题配置方法不尽相同，请参考配置文档。

打开配置文档，找到 comment，值改为 waline，之后找到 waline，serverURL填写上一步访问的域名（省略https://），保存后重新推送 Hexo，至此，配置完成，你可以通过 serverURL/ui来管理评论。

参考链接

https://waline.js.org/guide/get-started/
