3 个月过去了，谈谈我对小米澎湃 OS 的体验感受

发表于2024-07-10|更新于2024-07-10|体验
|浏览量:

前言

今年 4 月，小米向 Redmi Note 12T Pro 推送了首个澎湃 OS 的更新（OS1.0.1.0.ULHCNXM）。笔者收到更新通知之后，立即将更新包安装在了手机上。在这三个月（2024.4-2024.7）中，小米又向本机型推送了一次澎湃 OS 的更新（OS1.0.2.0.ULHCNXM）。那么，这两个澎湃 OS的版本，在 Redmi Note 12T Pro 上体验如何呢？我们来谈一谈——

遇到的问题

先说缺点。

在第一个版本 OS1.0.1.0.ULHCNXM 中，笔者遇到了一个极其严重的 BUG ——系统数据变得异常多。

多到什么程度呢，占了我大半的存储空间（见下图）。

一开始我并没有在意，以为是自己拍的视频和照片过多，或是某些 APP 缓存过量导致的。

直到我看到异常的系统数据数量时，我才发觉并不是以上原因导致的。

好在不就小米推送了第二个版本 OS1.0.2.0.ULHCNXM ，这个问题就被修复了。

但这个版本仍然有一个问题——音量调节有时会失效。有时我将手机音量置于 0 ，但手机仍在发出声音。

截至发稿，小米仍未推送更新来修复这个问题。我已经将这个问题进行了反馈，但是仍未得到任何回复。

还有，MIUI 中切换第三方输入法可以在任务栏中简便切换，而更新了澎湃 OS 后只能进入设置切换。

虽然澎湃 OS 支持了所谓“抠图”功能，但是只有部分图片支持，有的图片支持但也识别不到对象（见下图）。

做了跟没做似的，糊弄人呢？

不同的体验

相较于 MIUI ，小米的澎湃 OS 在实用性和美观性上都有了一些提升。

按照更新日志里的顺序，我来分别说说我的体验：

底层重构

性能和功耗相比 MIUI 相差不大，流畅度相比 MIUI 提升了不少。

信号方面，澎湃 OS 相比 MIUI 确实有了一点点的提升。

至于焕新存储，目前我的手机只进行了两次碎片处理（焕新存储需要一定条件才能触发，这里不阐释），所以我的感觉不大。

跨端智联

没体验过，没有发言权。

全域安全

没啥感觉。

生命感美学

该说不说，澎湃 OS 的美观程度比 MIUI 不知道高到哪里去了。

新的景深锁屏，我个人非常喜欢（见下图）。

新版的控制中心也变得简约美观了（见下图）。

但是有个地方我不太喜欢：某些胶囊通知变成了在通知栏上覆黑显示（见下图）。

除此之外，几乎没有可挑剔的地方了。

写在最后

这 3 个月下来，澎湃 OS 给我带来的惊喜与失望几乎是对等的。

澎湃 OS 在一款发布了近一年半的中端手机上仍然表现良好，希望小米能够加强对澎湃 OS 中 BUG 的修复及更新频率。

至于在澎湃 OS 中解 BootLoader ，我只能祝开发者们阖家幸福。
