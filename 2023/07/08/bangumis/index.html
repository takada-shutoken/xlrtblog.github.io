在 Hexo 博客中创建自己的追番列表

发表于2023-07-08|更新于2024-02-14|教程
|浏览量:

前言

使用 hexo-bilibili-bangumi 插件在 Hexo 博客中创建自己的追番列表。

项目介绍

为 Hexo 添加哔哩哔哩/Bangumi追番/追剧页面。

具体步骤

⒈ 安装

在 Hexo 安装目录下，执行

⒉ 添加配置

将以下配置写入 Hexo 的配置文件 _config.yml

其中 bangumi为追番的配置，cinema为追剧的配置。

下面列举出一些常用参数的使用方法：

source为数据来源，使用哔哩哔哩填写 bili，使用番组计划填写 bgm。追剧仅支持哔哩哔哩数据源。

bgmInfoSources为使用番组计划数据源时用到的 API ，具体参考https://github.com/bangumi/api/

vmid为哔哩哔哩账号的用户 UID 或番组计划的用户 ID （番组计划用户 ID 需自己设定）。

title为页面标题。

quote为页面介绍。

lazyload为页面懒加载开关，如与主题冲突请设置为 false。

⒊ 部署

使用 hexo bangumi -u更新追番数据，使用 hexo bangumi -u更新追剧数据。

执行完毕后正常进行生成部署。

使用 hexo bangumi -d删除追番数据，使用 hexo bangumi -d删除追剧数据。

部署完毕后，在 域名/bangumis查看追番列表，在 域名/cinemas查看追剧列表。
